Instrucciones de uso

  1. Ingresa el nombre de tu amigo. (No está permitido ingresar números)
  2. Presiona el botón Añadir amigo cada vez que termines de ingresar un nombre o presiona la tecla enter.
  3. Presiona Sortear amigo para realizar el sorteo,o
  4. Presiona Reiniciar sorteo si deseas borrar todos los datos ingresados.
